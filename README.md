# animaciones_css
Nöbe ☁️

Proyecto desarrollado para practicar CSS avanzado, diseño responsive y animaciones utilizando únicamente HTML y CSS.

Descripción

Nöbe es una landing page inspirada en distintos tipos de nubes. El proyecto utiliza una interfaz visual minimalista con efectos hover, glassmorphism, layouts adaptativos y una galería responsiva construida con Flexbox y CSS Grid.

El objetivo principal fue aplicar conceptos de:

* Responsive Design
* CSS Grid
* Flexbox
* Media Queries
* Transiciones y animaciones CSS
* Glassmorphism
* Organización visual y composición

⸻

Tecnologías utilizadas

* HTML5
* CSS3
* Font Awesome
* Google Fonts

⸻

Características

* Diseño adaptable para móvil, tablet y desktop
* Navegación responsive
* Cards interactivas con efectos hover
* Distribución dinámica mediante CSS Grid
* Estética visual basada en transparencias y desenfoque
* Uso de imágenes externas como fondos
